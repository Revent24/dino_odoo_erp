


Вот список самых полезных встроенных виджетов, которые ты можешь прописать прямо сейчас в атрибут widget="...".

1. Текстовые и строковые
char — обычное поле (по умолчанию).

text — многострочное поле.

html — редактор с форматированием (WYSIWYG).

copy_clipboard — добавляет кнопку "Скопировать" рядом с текстом.

email — делает ссылку mailto:.

phone — делает активную ссылку для звонка (VoIP).

url — кликабельная ссылка.

2. Числа и Деньги
monetary — добавляет значок валюты (требует поле currency_id в модели).

percentage — отображает 0.5 как 50%.

progressbar — полоска прогресса (обычно для полей от 0 до 100).

float_time — показывает число как часы:минуты (например, 1.5 станет 01:30).

statinfo — используется в Smart Buttons (верхние кнопки), показывает число и подпись жирным.

3. Выбор и Статусы
statusbar — всем известная полоска этапов вверху формы (Черновик -> В работе -> Готово).

selection — обычный выпадающий список.

radio — выбор точками (радиокнопки).

selection_badge — выбор в виде красивых "таблеток" (pill badges), часто используют вместо radio.

priority — звездочки (избранное), обычно для полей типа Selection.

boolean_toggle — вместо галочки (checkbox) рисует переключатель (тумблер) iPhone-стайл.

4. Связи (Relational)
many2one — стандартный выбор с поиском.

many2many_tags — теги (цветные плашки).

many2many_tags_avatar — теги, но с иконкой юзера внутри (как в задачах, когда назначаешь людей).

many2many_checkboxes — список с галочками (если вариантов мало).

handle — специальный виджет для поля sequence, позволяет перетаскивать строки в списке (drag & drop).

5. Файлы и Картинки
image — отображает картинку (с кнопкой редактирования по клику).

binary — кнопка скачивания файла.

pdf_viewer — встраивает просмотр PDF прямо в форму (удобно для счетов).

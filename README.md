# Dino ERP Module

This repository contains the `dino_erp` Odoo module extracted from the monorepo.

Contents:
- core/ — core models/views
- partners/ — partners models and views
- stock/ — stock models and views

Usage:
- Install as an Odoo addon (place it in addons path)
- Update module: `./odoo-bin -d <db> -i dino_erp`
